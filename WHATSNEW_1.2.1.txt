================================================================================
                           KIDUYU TV - VERSION 1.2.1
                          WHAT'S NEW / CHANGELOG
================================================================================

Release Date: April 7, 2026

================================================================================
                              NEW FEATURES
================================================================================

1. ENHANCED CAST DETAIL SCREEN
   - Added cast member biography (overview) display
   - Biography is shown in the fixed header section for quick reference
   - Click on biography to view the complete text in a dedicated dialog

2. IMPROVED HEADER LAYOUT
   - Header section is now fixed and does not scroll with the content
   - Header occupies 35% of the screen height for optimal visibility
   - Cast member profile image, name, and truncated biography are displayed
   - Responsive design adapts to different screen sizes

3. NEW FILMOGRAPHY GRID VIEW
   - Movies and TV shows are now displayed in a unified responsive grid
   - Grid layout automatically adjusts columns based on screen width
   - Media type badge (Movie/TV) is displayed on each item card
   - Clean, consistent presentation for all filmography items

4. ENHANCED TV REMOTE NAVIGATION
   - Full D-pad navigation support throughout the Cast Detail Screen
   - Visual focus indicators on all interactive elements
   - Smooth focus transitions between different sections
   - Optimized for comfortable TV remote control usage

5. INTERACTIVE BIOGRAPHY DIALOG
   - Truncated biography preview shows first 3 lines in header
   - Full biography accessible via focused click
   - Dialog has constrained height (200dp max) for better UX
   - Scrollable content within dialog for long biographies

================================================================================
                              IMPROVEMENTS
================================================================================

- Streamlined data fetching with parallel API calls
- Better state management for screen transitions
- Improved visual hierarchy with clear content separation
- Enhanced error handling and loading states
- Optimized layout performance for large filmography lists

================================================================================
                              TECHNICAL UPDATES

- Added PersonDetail data model for person-specific API responses
- Extended TmdbApiService with person details endpoint
- Updated TmdbRepository with new data fetching method
- Implemented CastDetailViewModel with concurrent data loading

================================================================================
                           KIDUYU TV - VERSION 1.2.2
                          WHAT'S NEW / CHANGELOG
================================================================================

Release Date: April 7, 2026

================================================================================
                              NEW FEATURES
================================================================================

1. ENHANCED "MY LIST" SCREEN WITH TABBED LAYOUT
   - Introduced a new tabbed interface for the "My List" screen.
   - Content is now categorized into distinct tabs: Movies, TV Shows, Companies, Networks, and Cast.
   - Provides a cleaner and more organized way to manage saved content.

2. SAVE COMPANIES AND NETWORKS TO "MY LIST"
   - Added functionality to save Production Companies and Networks to your personalized "My List."
   - A "+ Save to List" button is now available on the Media List screen for companies and networks.
   - Saved companies and networks are displayed in dedicated tabs within "My List."

3. SAVE CAST MEMBERS TO "MY LIST"
   - Implemented the ability to save individual Cast Members to "My List."
   - A "+ Save to List" button is now available on the Cast Detail screen.
   - Saved cast members are displayed in a dedicated tab within "My List."

4. LOGO DISPLAY FOR COMPANIES AND NETWORKS
   - "My List" now displays official logos for saved companies and networks.
   - If a logo is not available, the system gracefully falls back to displaying the name.

================================================================================
                              IMPROVEMENTS
================================================================================

- Improved navigation from "My List" to Company, Network, and Cast detail screens.
- Enhanced data persistence for saved cast members, including character and knownForDepartment information.
- Fixed clickability issues for Movie and TV Show cards within the "My List" screen.
- Optimized header layout in Cast Detail Screen for better visibility of the save button.

================================================================================
                              TECHNICAL UPDATES
================================================================================

- Updated `MyListManager` to handle new media types (company, network, cast).
- Modified `SavedMediaEntity` and `DatabaseManager` to store additional cast member details.
- Refactored `MediaListViewModel` and `CastDetailViewModel` to integrate saving logic.
- Updated `MyListScreen.kt` to implement `ScrollableTabRow` and categorized content display.
- Corrected `NavGraph.kt` to ensure proper navigation to Cast Detail with full `CastMember` data.
